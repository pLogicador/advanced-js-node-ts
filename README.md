# Repository for JavaScript and TypeScript Resources

Welcome to my repository where I catalog materials and exercises used during my JavaScript and TypeScript learning journey. This repository serves as a collection of files and resources employed throughout the course, covering a wide range of topics from fundamental to advanced concepts. The content spans both front-end and back-end development, exploring various technologies and frameworks.

## Key Learnings

- **Foundations: Mastery of Vanilla JavaScript (ES6+) for both front-end and back-end development.
- **Node, Express, and MongoDB: Building robust back-end applications with Node.js, Express, and MongoDB (NoSQL).
- **Web Development Tools: Understanding and utilizing Webpack and Babel for modern JavaScript development.
- **Security Practices: Implementation of secure coding practices and deployment on Linux servers with NGINX.
- **TypeScript Integration: Exploring TypeScript, a statically-typed superset of JavaScript, for enhanced development.
- **React JS: Creating dynamic and responsive user interfaces with React JS, incorporating React Hooks and Redux.
- **Database Management: Efficient handling of both SQL (MySQL/MariaDB) and NoSQL (MongoDB) databases.
- **API Development: Construction of RESTful APIs with Node, Express, and Sequelize, incorporating JWT authentication.
- **Testing Strategies: Implementing automated tests using Jest for unit and integration testing.
- **Design Patterns and Principles: Understanding SOLID principles and applying design patterns (GoF).
- **NextJS + Strapi: Development of SSR, SSG, and ISR applications with NextJS and Strapi.
- **Bonus Content: Exploration of regular expressions, HTML5, CSS3, and additional tools for a holistic skill set.
